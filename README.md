# AI Sales Analytics Project

##  Overview
This project demonstrates an end-to-end data analytics pipeline using Python, ETL processes, and Tableau. 
The goal is to transform raw sales data into meaningful business insights and visualize them using dashboards.

---

##  Tools & Technologies
- Python (Pandas)
- SQL (basic concepts)
- Tableau (Data Visualization)
- CSV (Data Source)
- AI-style insights (basic analysis)

---

##  Project Workflow
1. Data Extraction – Loaded raw sales data from CSV file(kiggle)  
2. Data Cleaning – Removed null values and duplicates using Python  
3. Data Transformation – Standardized column names and structure  
4. Data Analysis – Calculated total sales, top products, and regional performance  
5. Visualization – Built interactive dashboard in Tableau  
6. Insights – Generated business insights using simple AI-style logic  

---

##  Key Insights
- Identified top-performing products  
- Analyzed regional sales performance  
- Calculated total revenue trends  

---

##  Project Files
- `sales_data.csv` → Raw dataset
- `clean_sales_data.csv` → Cleaned dataset  
- `analysis.ipynb` → Python ETL + analysis code
